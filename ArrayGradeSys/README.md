# ArrayGradeSys - 基于数组的学生成绩管理系统

一个用C语言编写的简单学生成绩管理系统，使用二维数组存储学生信息，提供成绩录入、修改、删除、查询、排序等功能。

## 功能特性

- 学生成绩录入
- 学生成绩修改
- 学生成绩删除
- 学生成绩查询
- 学生成绩打印显示
- 学生成绩文件保存与读取
- 成绩排序（支持三种算法）
  - 冒泡排序
  - 插入排序
  - 选择排序

## 系统要求

- C语言编译器（如GCC、Clang或Visual Studio）
- 支持标准C库的运行环境

## 代码结构

### 主要数据结构

```c
// 最大学生人数
#define MaxNum 100
// 实际学生人数
int actnum=6;
// 存放学生信息的二维数组
int stu[MaxNum][2] = {{22, 64}, {26, 69}, {25, 70}, {21, 88}, {23, 91}, {30, 75}};
```

每个学生信息占用数组的一行，其中：
- `stu[i][0]` 表示第i个学生的学号
- `stu[i][1]` 表示第i个学生的成绩

### 主要功能函数

1. `input_score()` - 录入学生信息
2. `modify_score()` - 修改学生成绩
3. `remove_score()` - 删除学生信息
4. `print_score()` - 打印所有学生成绩
5. `search_score()` - 查询指定学号的学生成绩
6. `save_score()` - 将成绩保存到文件
7. `read_score()` - 从文件读取成绩
8. `bubble_sort_by_score()` - 使用冒泡排序按成绩排序
9. `insertion_sort_by_score()` - 使用插入排序按成绩排序
10. `selection_sort_by_score()` - 使用选择排序按成绩排序

## 编译和运行

### Windows系统

使用Visual Studio或MinGW编译器：

```bash
gcc main.c -o ArrayGradeSys.exe
ArrayGradeSys.exe
```

### Linux/macOS系统

```bash
gcc main.c -o ArrayGradeSys
./ArrayGradeSys
```

## 使用说明

程序启动后会显示菜单选项：

```
 1. 学生成绩录入
 2. 学生成绩修改
 3. 学生成绩删除
 4. 学生成绩查询
 5. 学生成绩打印
 6. 学生成绩保存
 7. 学生成绩读取
 8. 按成绩升序(冒泡排序)
 9. 按成绩升序(插入排序)
10. 按成绩升序(选择排序)
 0. 退出
```

输入对应数字选择相应功能。

### 功能详解

1. **学生成绩录入** - 可连续录入学生信息，包括学号和成绩
2. **学生成绩修改** - 根据学号修改指定学生的成绩
3. **学生成绩删除** - 根据学号删除指定学生的信息
4. **学生成绩查询** - 根据学号查询并显示学生成绩
5. **学生成绩打印** - 显示所有学生的学号和成绩
6. **学生成绩保存** - 将当前学生信息保存到文件中
7. **学生成绩读取** - 从文件中读取学生信息
8-10. **成绩排序** - 使用不同排序算法按成绩进行升序排列

## 文件操作

程序在保存数据时会在D盘根目录下创建或追加写入`score.txt`文件，格式为每行一个学生信息：`学号,成绩`。

## 注意事项

1. 系统最多支持100名学生信息管理
2. 默认初始化了6名学生的测试数据
3. 文件操作路径固定为D盘，请确保有相应权限或根据需要修改代码中的路径
4. 排序功能会对当前所有学生记录按成绩进行升序排列
