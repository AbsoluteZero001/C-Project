# LinkedGradeSys - 学生成绩管理系统

一个基于C语言开发的简单学生成绩管理系统，采用链表数据结构实现，具备基本的成绩录入、查询、修改、删除等功能。

## 功能特点

- 用户登录验证
- 成绩信息录入（学号、姓名、性别、成绩）
- 根据学号查询成绩
- 修改已有成绩记录
- 删除成绩记录
- 显示所有学生成绩
- 数据持久化存储（文件读写）

## 技术特性

- 使用链表数据结构管理学生信息
- 按学号排序存储学生记录
- 支持从文件读取和保存数据
- 控制台交互界面

## 系统架构

项目由以下几个模块组成：

- `main.c`: 主程序入口，负责流程控制
- `service.c/h`: 登录服务模块
- `util.c/h`: 核心业务逻辑模块，包括增删改查等操作

## 使用方法

1. 编译程序：
   ```
   gcc main.c service.c util.c -o LinkedGradeSys
   ```

2. 运行程序：
   ```
   ./LinkedGradeSys
   ```

3. 默认登录凭据：
   - 用户名：admin
   - 密码：abc123

4. 登录成功后即可使用各项功能：
   - 1 - 录入成绩
   - 2 - 查询成绩
   - 3 - 修改成绩
   - 4 - 删除成绩
   - 5 - 打印所有成绩
   - 6 - 保存成绩到文件
   - 7 - 从文件读取成绩
   - 0 - 退出系统

## 注意事项

- 程序默认从 D:\LinkArrayData.txt 读取和保存数据
- 学生信息按照学号从小到大排序存储
- 最多允许3次登录尝试

## 开发环境

- 编程语言：C语言
- 开发工具：Dev-C++ 或其他支持C语言的IDE
- 操作系统：Windows
