# TeaMenuMaster 茶饮菜单管理系统

一个使用C语言开发的简单茶饮菜单管理系统，包含后台管理和客户点餐两个模块。

## 项目概述

TeaMenuMaster是一个控制台应用程序，提供两种模式：
1. 后台管理系统 - 用于管理茶饮菜单
2. 客户点餐系统 - 供客户浏览和点餐

该系统具有清晰的菜单界面和彩色输出，提升用户体验。

## 功能特性

### 后台管理系统
- 查看所有茶饮
- 设置茶饮信息
- 删除茶饮
- 存储数据到文件
- 从文件加载茶饮数据

### 客户点餐系统
- 查看菜单
- 点餐
- 查看订单
- 取消订单
- 保存订单
- 加载菜单

## 项目结构

```
.
├── main.c              # 主程序入口
├── menu.h              # 菜单函数声明头文件
├── menuAdmin.c         # 后台管理功能实现
├── menuClient.c        # 客户端点餐功能实现
├── menuPrint.c         # 界面打印功能实现
└── menuPrint.h         # 打印函数声明头文件
```

## 编译和运行

### Windows环境

使用Dev-C++或其他支持的IDE直接打开项目文件进行编译运行。

或者使用命令行：

```bash
gcc main.c menuAdmin.c menuClient.c menuPrint.c -o TeaMenuMaster
./TeaMenuMaster
```

### Linux/Mac环境

```bash
gcc main.c menuAdmin.c menuClient.c menuPrint.c -o TeaMenuMaster
./TeaMenuMaster
```

## 使用说明

1. 运行程序后，会提示选择系统类型：
   - 输入 `1` 进入后台管理系统
   - 输入 `2` 进入客户点餐系统
   - 其他输入将退出程序

2. 根据屏幕提示进行相应操作

3. 每次操作完成后按任意键继续

## 界面特色

程序使用ANSI转义序列为界面添加了颜色效果：
- 标题采用绿色粗体显示
- 子标题采用青色粗体显示
- 选项编号为黄色，选项文字为白色
- 分割线为青色

## 开发环境

- 编程语言：C语言
- 开发工具：Dev-C++
- 操作系统：跨平台（Windows/Linux/Mac）
